### context
I was responsible for infrastructure and systems across headquarters, warehouse, and 25 stores, supporting more than 100 users with no real handover and a very weak starting point.

### responsibilities
I reverse-engineered the environment while keeping daily operations running. That included network segmentation, rack and server organization, Windows Server and Active Directory setup, shared data structure, Linux and Windows administration, recovery scripting, alerts, supplier coordination, cost reduction, and continuous support across sites.

### scope
The work also went into SAP-related systems, HANA queries and procedures, reconciliation, reporting, POS refactoring, service recovery, alerts, and automation whenever repeated manual work started becoming a waste of time.

### result
Over time the company moved from a fragile and disorganized setup to a much more controlled and reliable environment.
