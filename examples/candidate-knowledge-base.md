# About Me

I am a Systems Engineer focused on building and stabilizing IT infrastructure from scratch. I evolved from bench hardware diagnostics to managing entire IT departments, connecting the physical layer to automation. My goal is to solve business problems through scalable architectures and data integration.

# Work Experience

I worked as IT Systems Engineer at a retail company with about 100 users across headquarters, a warehouse and 25 stores. When I joined in 2022, there was no IT foundation at all. No handover, no documentation. Windows 2000 servers, Windows Vista and 7 PCs. The network was flat, meaning SAP, internal systems and public WiFi were all on the same segment. I was the only IT person.

I started by reverse-engineering everything while providing daily support. Then I redesigned the network from flat topology to segmented VLANs across all locations. I built the server room from scratch, sourced hardware, mounted racks, organized cabling. Deployed Windows Server 2022 with Active Directory, domain controller, departmental file shares and GPOs. Set up a UniFi controller on a virtual machine for the access points. Implemented recovery scripts for both Windows and Linux via PowerShell and crontab.

Once the infrastructure was stable, I moved into automation and data. Built a Python FastAPI backend for real-time SAP HANA sync. Created Python ETL pipelines for automated daily sales reports delivered by email to management. Developed a C# .NET application to reconcile POS data with SAP B1. Refactored POS components in SAPUI5 XML views and JavaScript controllers to reduce data entry errors. Automated the SAFT-PT tax compliance system for multi-store fiscal processing. Optimized SAP HANA with complex SQL queries and stored procedures.

I also negotiated vendor contracts and reduced operational costs. Modernized all legacy equipment across the 25 retail stores.

Before that, I worked for six years at DuploClique in Póvoa de Varzim. I started as an intern after completing my IT certification and stayed for the experience. I was the only technical person there.

Early on, I did bench repairs at component level. Micro-soldering, hardware replacement, upgrades on computers, laptops, smartphones, tablets, servers. Built the first in-store file server.

Later I moved to systems administration and web development. Managed the full stack alone: Apache, PHP, MySQL, DNS zones, SSL certificates, Cloudflare CDN and WAF. Built and maintained WordPress and WooCommerce sites for external clients. Custom themes, security hardening via hooks instead of plugins, payment gateway integration. Stock management: converted supplier XLSX files to CSV and synced via WooCommerce REST API.

Migrated all infrastructure from on-premises bare-metal to AWS Lightsail. Everything through SFTP and terminal. Wrote Bash and Python scripts for backups, maintenance, monitoring.

I also handled technical client consultations, cost management with suppliers, and project scoping for external infrastructure and web development work.
